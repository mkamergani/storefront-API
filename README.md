## Installation

Install my-project with npm

```bash
  npm install
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```

## Building, Formatting, Linting 

To run tests, run the following command

```bash
  npm run build
```

```bash
  npm run prettier
```
